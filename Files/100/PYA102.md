# PYA102 浮點數格式化輸出

**類別：第1類：基本程式設計**

## 題目說明
讓使用者輸入四個浮點數 a, b, c, d，分別以靠右及靠左對齊（欄寬7，小數2位）各輸出兩行，以 | 作為邊界。

## 範例輸入
```
1.5
2.3
4.7
8.1
```

## 輸出結果
```
|   1.50    2.30|
|   4.70    8.10|
|1.50    2.30   |
|4.70    8.10   |
```

## 程式碼
```python
a=float(input())
b=float(input())
c=float(input())
d=float(input())
print("|%7.2f %7.2f|"%(a,b))
print("|%7.2f %7.2f|"%(c,d))
print("|{:<7.2f} {:<7.2f}|".format(a,b))
print("|{:<7.2f} {:<7.2f}|".format(c,d))
```

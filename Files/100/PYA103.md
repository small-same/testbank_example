# PYA103 字串格式化輸出

**類別：第1類：基本程式設計**

## 題目說明
讓使用者輸入四個字串 a, b, c, d，分別以靠右（>）和靠左（<）對齊方式各輸出兩行，欄寬為10，以 | 作為邊界。

## 範例輸入
```
Hello
World
Python
Code
```

## 輸出結果
```
|     Hello      World|
|    Python       Code|
|Hello      World     |
|Python     Code      |
```

## 程式碼
```python
a=input()
b=input()
c=input()
d=input()
print('|{:>10s} {:>10s}|'.format(a,b))
print('|{:>10s} {:>10s}|'.format(c,d))
print('|{:<10s} {:<10s}|'.format(a,b))
print('|{:<10s} {:<10s}|'.format(c,d))
```

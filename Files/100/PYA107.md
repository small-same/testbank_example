# PYA107 數值計算

**類別：第1類：基本程式設計**

## 題目說明
讓使用者輸入五個數字，輸出這五個數字、其總和與平均，輸出至小數點後第一位。

## 範例輸入
```
1
2
3
4
5
```

## 輸出結果
```
1 2 3 4 5
Sum = 15.0
Average = 3.0
```

## 程式碼
```python
l=[0]*5
l[0]=eval(input())
l[1]=eval(input())
l[2]=eval(input())
l[3]=eval(input())
l[4]=eval(input())
print(l[0],l[1],l[2],l[3],l[4])
print('Sum = %.1f'%sum(l))
print('Average = %.1f'%(sum(l)/len(l)))
```

# PYA105 矩形面積計算

**類別：第1類：基本程式設計**

## 題目說明
讓使用者輸入矩形的高 h 和寬 w，計算並輸出高度、寬度、周長與面積，輸出至小數點後第二位。

## 範例輸入
```
3.0
4.0
```

## 輸出結果
```
Height = 3.00
Width = 4.00
Perimeter = 14.00
Area = 12.00
```

## 程式碼
```python
h=eval(input())
w=eval(input())
print('Height = %.2f'%h)
print('Width = %.2f'%w)
print('Perimeter = %.2f'%(w*2+h*2))
print('Area = %.2f'%(w*h))
```

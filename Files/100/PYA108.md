# PYA108 座標距離計算

**類別：第1類：基本程式設計**

## 題目說明
讓使用者輸入兩點座標 (x1,y1) 和 (x2,y2)，計算並輸出兩點的歐式距離，輸出至小數點後第四位。

## 範例輸入
```
0
0
3
4
```

## 輸出結果
```
( 0 , 0 )
( 3 , 4 )
Distance = 5.0000
```

## 程式碼
```python
x1=eval(input())
y1=eval(input())
x2=eval(input())
y2=eval(input())
print(f'( {x1} , {y1} )')
print(f'( {x2} , {y2} )')
print('Distance = %.4f'% ((x1-x2)**2+(y1-y2)**2)**0.5)
```

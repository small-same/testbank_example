# PYA104 圓形面積計算

**類別：第1類：基本程式設計**

## 題目說明
讓使用者輸入圓的半徑 r，計算並輸出半徑、周長與面積，輸出至小數點後第二位。

## 範例輸入
```
5
```

## 輸出結果
```
Radius = 5.00
Perimeter = 31.42
Area = 78.54
```

## 程式碼
```python
import math
r=eval(input(""))
print("Radius = %.2f"%r)
print("Perimeter = %.2f"%(r*2*math.pi))
print("Area = %.2f"%(r*r*math.pi))
```

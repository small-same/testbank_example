# PYA101 整數格式化輸出

**類別：第1類：基本程式設計**

## 題目說明
讓使用者輸入四個字串 a, b, c, d，分別以靠右（>）和靠左（<）對齊方式各輸出兩行，欄寬為5，以 | 作為邊界。

## 範例輸入
```
Tom
Jerry
Alice
Bob
```

## 輸出結果
```
|  Tom Jerry|
|Alice   Bob|
|Tom   Jerry|
|Alice Bob  |
```

## 程式碼
```python
a=input()
b=input()
c=input()
d=input()
print('|{:>5s} {:>5s}|'.format(a,b))
print('|{:>5s} {:>5s}|'.format(c,d))
print('|{:<5s} {:<5s}|'.format(a,b))
print('|{:<5s} {:<5s}|'.format(c,d))
```

# PYA610 平均溫度

**類別：第6類：串列**

## 題目說明
讓使用者輸入四週各三天的溫度，計算並輸出平均溫度（至小數點後第二位）及最高、最低溫度。

## 範例輸入
```
25
26
27
28
29
30
22
23
24
31
32
33
```

## 輸出結果
```
Week 1:
Day 1:Day 2:Day 3:Week 2:
Day 1:Day 2:Day 3:Week 3:
Day 1:Day 2:Day 3:Week 4:
Day 1:Day 2:Day 3:Average: 27.50
Highest: 33
Lowest: 22
```

## 程式碼
```python
L=[]
for w in range(1,4+1):
    print("Week %d:"%w)
    for D in range(1,3+1):
        x=eval(input("Day %d:"%D))
        L.append(x)
A=sum(L)/len(L)
print("Average: %.2f"%A)
print("Highest:",max(L))
print("Lowest:",min(L))
```

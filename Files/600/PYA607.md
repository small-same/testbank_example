# PYA607 成績計算

**類別：第6類：串列**

## 題目說明
讓使用者輸入三位學生各五筆成績，計算並輸出每位學生的總分及平均分數（平均至小數點後第二位）。

## 範例輸入
```
80
90
70
85
95
75
85
65
90
80
60
70
80
75
85
```

## 輸出結果
```
The 1st student:
The 2nd student:
The 3rd student:
Student 1
#Sum 420
#Average 84.00
Student 2
#Sum 395
#Average 79.00
Student 3
#Sum 370
#Average 74.00
```

## 程式碼
```python
n=["1st","2nd","3rd"]
t=[[0 for j in range(5)] for i in range(3)]
for i in range(3):
    print("The {} student:".format(n[i]))
    for j in range(5):
        t[i][j]=int(input())
for i in range(3):
    print("Student %d"%(i+1))
    print("#Sum %d"%(sum(t[i])))
    print("#Average %.2f"%(sum(t[i])/5))
```

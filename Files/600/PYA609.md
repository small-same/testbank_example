# PYA609 矩陣相加

**類別：第6類：串列**

## 題目說明
讓使用者建立兩個2×2的矩陣，輸出兩個矩陣的內容及其相加結果。

## 範例輸入
```
1
2
3
4
5
6
7
8
```

## 輸出結果
```
Enter matrix 1:
[1, 1]: [1, 2]: [2, 1]: [2, 2]: Enter matrix 2:
[1, 1]: [1, 2]: [2, 1]: [2, 2]: Matrix 1:
1 2 
3 4 
Matrix 2:
5 6 
7 8 
Sum of 2 matrices:
6 8 
10 12
```

## 程式碼
```python
L1 = [[0 for i in range(2)] for j in range(2)]
L2 = [[0 for i in range(2)] for j in range(2)]
print('Enter matrix 1:')
for i in range(2):
    for j in range(2):
        L1[i][j] = int(input('[%d, %d]: '%(i+1,j+1)))
print('Enter matrix 2:')
for i in range(2):
    for j in range(2):
        L2[i][j] = int(input('[%d, %d]: '%(i+1,j+1)))
print('Matrix 1:')
for i in range(2):
    for j in range(2):
        print(L1[i][j],end=' ')
    print()
print('Matrix 2:')
for i in range(2):
    for j in range(2):
        print(L2[i][j],end=' ')
    print()
print('Sum of 2 matrices:')
for i in range(2):
    for j in range(2):
        print((L1[i][j]+L2[i][j]),end=' ')
    print()
```

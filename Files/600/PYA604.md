# PYA604 眾數

**類別：第6類：串列**

## 題目說明
讓使用者輸入十個整數（假設只有一個眾數），輸出眾數及其出現的次數。

## 範例輸入
```
1
2
3
3
3
4
5
6
7
8
```

## 輸出結果
```
3
3
```

## 程式碼
```python
m=[0]*100
for i in range(10):
    n=int(input())
    m[n]+=1
print(m.index(max(m)))
print(max(m))
```

### 解法 2
```python
m=[0]*100
for i in range(10):
    n=int(input())
    m[n]+=1
for i in range(0,100):
    if(max(m)==m[i]):
        print(i)
        print(max(m))
```

# PYA602 撲克牌總和

**類別：第6類：串列**

## 題目說明
讓使用者輸入52張牌中的5張，計算並輸出總和。（J=11，Q=12，K=13，A=1）

## 範例輸入
```
A
J
Q
K
5
```

## 輸出結果
```
42
```

## 程式碼
```python
t=0
for i in range(5):
    n=str(input())
    if(n=='J'): t+=11
    elif(n=='Q'): t+=12
    elif(n=='K'): t+=13
    elif(n=='A'): t+=1
    else : t+=int(n)
print(t)
```

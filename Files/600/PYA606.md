# PYA606 二維串列行列數

**類別：第6類：串列**

## 題目說明
讓使用者輸入 rows 和 cols，建立二維串列，元素 [row][col] = col - row，並以欄寬4格式呼叫 compute() 輸出。

## 範例輸入
```
2
3
```

## 輸出結果
```
0   1   2
  -1   0   1
```

## 程式碼
```python
def compute(r,c):
    for i in range(r):
        for j in range(c):
            print("%4d"%(j-i), end='')
        print()
r = int(input())
c = int(input())
compute(r,c)
```

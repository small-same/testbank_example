# PYA505 依參數格式化輸出

**類別：第5類：函式**

## 題目說明
讓使用者輸入字元 a、個數 x、列數 y，呼叫 compute(a, x, y) 函式，一列印出 x 個 a 字元，共印 y 列（每字元後有一空格）。

## 範例輸入
```
*
4
3
```

## 輸出結果
```
* * * * 
* * * * 
* * * *
```

## 程式碼
```python
def compute(a,x,y):
    for i in range(y):
        for j in range(x):
            print('%c '%a,end='')
        print()
a = input()
x = eval(input())
y = eval(input())
compute(a,x,y)
```

# PYA508 最大公因數

**類別：第5類：函式**

## 題目說明
讓使用者以逗號分隔輸入兩個正整數（如 12,8），呼叫 compute(x, y) 回傳最大公因數（GCD）並輸出。

## 範例輸入
```
12,8
```

## 輸出結果
```
4
```

## 程式碼
```python
import math
def compute(x,y):
    return math.gcd(x,y)
x,y=eval(input())
print(compute(x,y))
```

### 解法 2
```python
def compute(x,y):
    while(y!=0):
        temp=y
        y=x%y
        x=temp
    print(x)
x,y=eval(input())
compute(x,y)
```

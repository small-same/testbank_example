# PYA502 乘積

**類別：第5類：函式**

## 題目說明
將使用者輸入的兩個整數傳遞給 compute(x, y) 函式，回傳並輸出兩數的乘積。

## 範例輸入
```
3
4
```

## 輸出結果
```
12
```

## 程式碼
```python
def compute(x,y):
    print(x*y)
compute(int(input()),int(input()))
```

### 解法 2
```python
def compute(x,y):
    return x*y
x = eval(input())
y = eval(input())
print(compute(x,y))
```

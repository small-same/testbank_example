# PYA510 費氏數列

**類別：第5類：函式**

## 題目說明
讓使用者輸入正整數 num（num≥2），呼叫 compute() 函式輸出費氏數列前 num 個數值。（F₀=0，F₁=1，Fₙ=Fₙ₋₁+Fₙ₋₂）

## 範例輸入
```
8
```

## 輸出結果
```
0 1 1 2 3 5 8 13
```

## 程式碼
```python
def compute(n):
    if n > 1:
        return compute(n-1) + compute(n-2)
    return n
n=int(input())
for i in range(n):
    print(compute(i), end = ' ')
```

### 解法 2
```python
def compute(n):
    curr = 1
    pre = 0
    print(0,1,end=' ')
    for i in range(2,n):
        temp = curr+pre
        print(temp,end=' ')
        pre = curr
        curr = temp
num = eval(input())
compute(num)
```

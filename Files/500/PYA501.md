# PYA501 訊息顯示

**類別：第5類：函式**

## 題目說明
呼叫函式 compute()，讓使用者輸入系別（Department）、學號（Student ID）和姓名（Name）並顯示這些訊息。

## 範例輸入
```
Computer Science
S10001
Alice
```

## 輸出結果
```
Department: Computer Science
Student ID: S10001
Name: Alice
```

## 程式碼
```python
def compute():
    department = input()
    studentid = input()
    name = input()
    print('Department:',department)
    print('Student ID:',studentid)
    print('Name:',name)
compute()
```

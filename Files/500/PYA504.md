# PYA504 次方計算

**類別：第5類：函式**

## 題目說明
讓使用者輸入兩個整數，呼叫函式 compute(a, b)，回傳 a^b 的值並輸出。

## 範例輸入
```
2
8
```

## 輸出結果
```
256
```

## 程式碼
```python
def compute(x,y):
    print(x**y)
compute(int(input()),int(input()))
```

### 解法 2
```python
def compute(a,b):
    return pow(a,b)
x = eval(input())
y = eval(input())
print(compute(x,y))
```

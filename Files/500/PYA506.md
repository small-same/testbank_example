# PYA506 一元二次方程式

**類別：第5類：函式**

## 題目說明
讓使用者輸入三個係數 a, b, c（代表 ax²+bx+c=0），呼叫 compute() 函式求解；無解則輸出 'Your equation has no root.'

## 範例輸入
```
1
-3
2
```

## 輸出結果
```
2.0, 1.0
```

## 程式碼
```python
def compute(a,b,c):
    temp=b**2-4*a*c
    if(temp<0): print('Your equation has no root.')
    elif(temp==0) :
        ans=-b/2*a
        print('%.1f'%ans)
    elif(temp>0) :
        print('%.1f, %.1f'%((-b+temp**0.5)/(2*a),(-b-temp**0.5)/(2*a)))
compute(int(input()),int(input()),int(input()))
```

### 解法 2
```python
def compute(a,b,c):
    q=b**2-4*a*c
    if q<0:
        print("Your equation has no root.")
    elif q==0:
        print(-b/2*a)
    else:
        q1=(-b+q**0.5)/(2*a)
        q2=(-b-q**0.5)/(2*a)
        print('{}, {}'.format(q1, q2))
a=int(input())
b=int(input())
c=int(input())
compute(a,b,c)
```

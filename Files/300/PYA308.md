# PYA308 迴圈位數加總

**類別：第3類：迴圈敘述**

## 題目說明
讓使用者輸入測試資料數量，接著對每個正整數，將其各位數相加並輸出結果。

## 範例輸入
```
2
123
456
```

## 輸出結果
```
Sum of all digits of 123 is 6
Sum of all digits of 456 is 15
```

## 程式碼
```python
for i in range(int(input())):
    n=str(input())
    a = [int(j) for j in list(str(n))]
    print("Sum of all digits of "+str(n)+" is "+str(sum(a)))
```

### 解法 2
```python
a=int(input())
for i in range(a):
    ans=0
    n=str(input())
    for i in range(0,len(n)):
        ans+=int(n[i])
    print("Sum of all digits of "+str(n)+" is "+str(ans))
```

### 解法 3
```python
n = eval(input())
for i in range(n):
    num = eval(input())
    t = num
    ans = 0
    while t!=0:
        ans+=(t%10)
        t=t//10
    print('Sum of all digits of',num,'is',ans)
```

# PYA307 乘法表

**類別：第3類：迴圈敘述**

## 題目說明
讓使用者輸入一個正整數 n（n<10），顯示 n×n 乘法表。每個運算子及運算元欄寬為2，乘積欄寬為4，靠左對齊不跳行。

## 範例輸入
```
3
```

## 輸出結果
```
1 * 1 = 1   2 * 1 = 2   3 * 1 = 3   
1 * 2 = 2   2 * 2 = 4   3 * 2 = 6   
1 * 3 = 3   2 * 3 = 6   3 * 3 = 9
```

## 程式碼
```python
n=int(input())
for i in range(1,n+1):
    for j in range(1,n+1):
        print('{} * {} = {:<4}'.format(j,i,(j*i)),end='')
    print()
```

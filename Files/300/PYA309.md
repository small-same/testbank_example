# PYA309 存款總額

**類別：第3類：迴圈敘述**

## 題目說明
讓使用者輸入本金、年收益率及月份數，顯示每個月的存款總額，四捨五入至小數點後第二位。

## 範例輸入
```
10000
5.75
3
```

## 輸出結果
```
Month 	 Amount
  1 	 10047.92
  2 	 10096.06
  3 	 10144.44
```

## 程式碼
```python
total = eval(input())
y = eval(input())
m = eval(input())
print('%s 	 %s' % ('Month', 'Amount'))
for i in range(1, m + 1):
    total += total * y / 1200
    print('%3d 	 %.2f' % (i, total))
```

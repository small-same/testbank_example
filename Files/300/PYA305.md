# PYA305 數字反轉

**類別：第3類：迴圈敘述**

## 題目說明
讓使用者輸入一個正整數，將此數值以反轉的順序輸出。

## 範例輸入
```
12345
```

## 輸出結果
```
54321
```

## 程式碼
```python
a=int(input())
if(a==0):
    print(0)
while (a!=0) :
    print(int(a%10),end="")
    a=a//10
```

### 解法 2
```python
a=str(input())
reversed_string=''.join(reversed(a))
print(reversed_string)
```

# PYA303 迴圈數值相乘

**類別：第3類：迴圈敘述**

## 題目說明
讓使用者輸入一個正整數（<100），以三角形方式輸出該數的相乘結果，欄寬為4，靠右對齊。

## 範例輸入
```
4
```

## 輸出結果
```
1
   2   4
   3   6   9
   4   8  12  16
```

## 程式碼
```python
a=eval(input())
c=0
for i in range(1,a+1):
    for j in range(1,i+1):
        c=i*j
        print("%4d"%(c),end='')
    print()
```

# PYA705 子集合與超集合

**類別：第7類：數組、集合、詞典**

## 題目說明
依序輸入五個、三個、九個整數分別存入 set1、set2、set3，判斷 set2 是否為 set1 的子集合，以及 set3 是否為 set1 的超集合。

## 範例輸入
```
1
2
3
4
5
2
3
4
1
2
3
4
5
6
7
8
9
```

## 輸出結果
```
Input to set1:
Input to set2:
Input to set3:
set2 is subset of set1: True
set3 is superset of set1: True
```

## 程式碼
```python
set1 = set()
set2 = set()
set3 = set()
print('Input to set1:')
for i in range(5):
    set1.add(eval(input()))
print('Input to set2:')
for i in range(3):
    set2.add(eval(input()))
print('Input to set3:')
for i in range(9):
    set3.add(eval(input()))
print('set2 is subset of set1:',str(set2.issubset(set1)))
print('set3 is superset of set1:',str(set3.issuperset(set1)))
```

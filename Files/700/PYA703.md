# PYA703 數組條件判斷

**類別：第7類：數組、集合、詞典**

## 題目說明
輸入字串存入數組（至少五個，以 end 結束），輸出數組、前三個元素和倒數三個元素。

## 範例輸入
```
apple
banana
cherry
date
elderberry
fig
end
```

## 輸出結果
```
('apple', 'banana', 'cherry', 'date', 'elderberry', 'fig')
('apple', 'banana', 'cherry')
('date', 'elderberry', 'fig')
```

## 程式碼
```python
TUPLE = ()
string = input()
while string != "end":
    TUPLE +=(string,)
    string = input()
print(TUPLE)
print(TUPLE[0:3])
print(TUPLE[-3:])
```

# PYA706 全字母句

**類別：第7類：數組、集合、詞典**

## 題目說明
輸入 k 筆句子，判斷每句是否為 Pangram（26個英文字母各至少出現一次），輸出 True 或 False。不區分大小寫。

## 範例輸入
```
2
The quick brown fox jumps over the lazy dog
Hello World
```

## 輸出結果
```
True
False
```

## 程式碼
```python
count= eval(input())
alpha=26
for i in range(count):
    sentence = input()
    set1 = set(sentence.lower())
    if ' ' in set1:
        set1.remove(' ')
    if len(set1) >= alpha:
        print('True')
    else:
        print('False')
```

### 解法 2
```python
n=int(input())
for i in range(n):
    a=[0]*26
    f=True
    temp=input()
    temp=temp.lower()
    for i in range(len(temp)):
        if(temp[i]!=' '):
            a[ord(temp[i])-ord('a')]+=1
    for i in range(len(a)):
        if(a[i]==0) : f=False
    if(f) : print('True')
    else : print('False')
```

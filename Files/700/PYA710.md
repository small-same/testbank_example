# PYA710 詞典搜尋

**類別：第7類：數組、集合、詞典**

## 題目說明
為詞典輸入資料（以 end 結束），再輸入一鍵值，檢視此鍵值是否存在於詞典中並輸出 True 或 False。

## 範例輸入
```
name
Alice
age
20
end
name
```

## 輸出結果
```
Key: Value: Key: Value: Key: Search key: True
```

## 程式碼
```python
d = {}
while True:
    key = input('Key: ')
    if key == 'end':
        break
    d[key] = input('Value: ')
search = input('Search key: ')
print(search in d.keys())
```

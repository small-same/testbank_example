# PYA704 集合條件判斷

**類別：第7類：數組、集合、詞典**

## 題目說明
輸入數個整數存入集合（-9999結束），輸出集合的長度、最大值、最小值、總和。

## 範例輸入
```
10
20
30
-9999
```

## 輸出結果
```
Length: 3
Max: 30
Min: 10
Sum: 60
```

## 程式碼
```python
set1 = set()
num = eval(input())
while num!=-9999:
    set1.add(num)
    num = eval(input())
print('Length:',len(set1))
print('Max:',max(set1))
print('Min:',min(set1))
print('Sum:',sum(set1))
```

### 解法 2
```python
s1=set()
while True :
    n=int(input())
    if(n==-9999) : break
    s1.add(n)
print('Length:',len(s1))
print('Max:',max(s1))
print('Min:',min(s1))
print('Sum:',sum(s1))
```

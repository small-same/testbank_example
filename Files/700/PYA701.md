# PYA701 串列數組轉換

**類別：第7類：數組、集合、詞典**

## 題目說明
輸入數個整數存入串列（-9999結束），轉換成數組後顯示數組及其長度、最大值、最小值、總和。

## 範例輸入
```
3
5
2
8
-9999
```

## 輸出結果
```
(3, 5, 2, 8)
Length: 4
Max: 8
Min: 2
Sum: 18
```

## 程式碼
```python
LIST = []
num = eval(input())
while num!=-9999:
    LIST.append(num)
    num = eval(input())
TUPLE = tuple(LIST)
print(TUPLE)
print('Length:',len(TUPLE))
print('Max:',max(TUPLE))
print('Min:',min(TUPLE))
print('Sum:',sum(TUPLE))
```

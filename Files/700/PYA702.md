# PYA702 數組合併排序

**類別：第7類：數組、集合、詞典**

## 題目說明
建立兩組數組（各以-9999結束），合併後從小到大排序，顯示排序前的數組和排序後的串列。

## 範例輸入
```
3
1
4
-9999
1
5
9
-9999
```

## 輸出結果
```
Create tuple1:
Create tuple2:
Combined tuple before sorting: (3, 1, 4, 1, 5, 9)
Combined list after sorting: [1, 1, 3, 4, 5, 9]
```

## 程式碼
```python
t1=()
print('Create tuple1:')
while True :
    n=int(input())
    if(n==-9999) : break
    t1+=(n,)
t2=()
print('Create tuple2:')
while True :
    n=int(input())
    if(n==-9999) : break
    t2+=(n,)
t1 += t2
print('Combined tuple before sorting:',t1)
LIST = list(t1)
LIST.sort()
print('Combined list after sorting:',LIST)
```

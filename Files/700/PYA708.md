# PYA708 詞典合併

**類別：第7類：數組、集合、詞典**

## 題目說明
輸入兩個詞典（以 end 結束鍵值輸入），合併後依 key 字母排序輸出，重複 key 由後者覆蓋。

## 範例輸入
```
name
Alice
age
20
end
city
Taipei
grade
A
end
```

## 輸出結果
```
Create dict1:
Key: Value: Key: Value: Key: Create dict2:
Key: Value: Key: Value: Key: age: 20
city: Taipei
grade: A
name: Alice
```

## 程式碼
```python
dc={}
print('Create dict1:')
while True :
    k=input('Key: ')
    if k=='end' : break
    dc[k]=input('Value: ')
print('Create dict2:')
while True :
    k=input('Key: ')
    if k=='end' : break
    dc[k]=input('Value: ')
for i in sorted(dc.keys()):
    print(i+": "+dc[i])
```

### 解法 2
```python
x={}
y={}
print('Create dict1:')
while True:
    key = input('Key: ')
    if key == 'end':
        break
    x[key] = input('Value: ')
print('Create dict2:')
while True:
    key = input('Key: ')
    if key == 'end':
        break
    y[key] = input('Value: ')
x.update(y)
for i in sorted(x.keys()):
    print(i+": "+x[i])
```

# PYA709 詞典排序

**類別：第7類：數組、集合、詞典**

## 題目說明
輸入一個顏色詞典（以 end 結束），根據 key 字母由小到大排序並輸出。

## 範例輸入
```
red
#FF0000
green
#00FF00
blue
#0000FF
end
```

## 輸出結果
```
Key: Value: Key: Value: Key: Value: Key: blue: #0000FF
green: #00FF00
red: #FF0000
```

## 程式碼
```python
d= {}
while True:
    key = input('Key: ')
    if key == 'end':
        break
    d[key] = input('Value: ')
for i in sorted(d.keys()):
    print(i+': '+d[i])
```

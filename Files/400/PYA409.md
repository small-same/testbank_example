# PYA409 得票數計算

**類別：第4類：進階控制流程**

## 題目說明
輸入五張選票（1=Nami，2=Chopper，其他=廢票），每票後印出目前得票數，最後印出當選人（同票則無人當選）。

## 範例輸入
```
1
1
2
1
2
```

## 輸出結果
```
Total votes of No.1: Nami = 1
Total votes of No.2: Chopper = 0
Total null votes = 0
Total votes of No.1: Nami = 2
Total votes of No.2: Chopper = 0
Total null votes = 0
Total votes of No.1: Nami = 2
Total votes of No.2: Chopper = 1
Total null votes = 0
Total votes of No.1: Nami = 3
Total votes of No.2: Chopper = 1
Total null votes = 0
Total votes of No.1: Nami = 3
Total votes of No.2: Chopper = 2
Total null votes = 0
=> No.1 Nami won the election.
```

## 程式碼
```python
count1 = 0
count2 = 0
null = 0
for i in range(5):
    vote = int(input())
    if vote == 1:
        count1 += 1
    elif vote == 2:
        count2 += 1
    else:
        null += 1
    print("Total votes of No.1: Nami = %d" % count1)
    print("Total votes of No.2: Chopper = %d" % count2)
    print("Total null votes = %d" % (null))
if count1 > count2:
    print("=> No.1 Nami won the election.")
elif count1 < count2:
    print("=> No.2 Chopper won the election.")
else:
    print("=> No one won the election.")
```

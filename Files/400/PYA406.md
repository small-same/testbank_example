# PYA406 不定數迴圈-BMI計算

**類別：第4類：進階控制流程**

## 題目說明
以不定數迴圈輸入身高與體重，計算BMI並輸出對應狀態（under weight / normal / over weight / fat）。輸入-9999結束。

## 範例輸入
```
170
60
-9999
```

## 輸出結果
```
BMI: 20.76
State: normal
```

## 程式碼
```python
while True:
    H = float(input())
    if H == -9999: break
    else:
        W = float(input())
        BMI = W / (H/100)**2
        print(f'BMI: {BMI:.2f}')
        if BMI < 18.5: print("State: under weight")
        if 18.5 <= BMI < 25: print("State: normal")
        if 25.0 <= BMI < 30: print("State: over weight")
        if 30 <= BMI: print("State: fat")
```

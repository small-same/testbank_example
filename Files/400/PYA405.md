# PYA405 不定數迴圈-分數等級

**類別：第4類：進階控制流程**

## 題目說明
以不定數迴圈輸入分數，根據對照表輸出GPA（90~100→A，80~89→B，70~79→C，60~69→D，0~59→E）。輸入-9999結束。

## 範例輸入
```
95
75
55
-9999
```

## 輸出結果
```
A
C
E
```

## 程式碼
```python
while True :
    n=int(input())
    if n==-9999: break
    if(n>0 and n<60):
        print('E')
    elif(n>=60 and n<70):
        print('D')
    elif(n>=70 and n<80):
        print('C')
    elif(n>=80 and n<90):
        print('B')
    elif(n>=90 and n<=100):
        print('A')
```

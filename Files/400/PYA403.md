# PYA403 倍數總和計算

**類別：第4類：進階控制流程**

## 題目說明
讓使用者輸入兩個正整數 a、b（a≤b），輸出 a 到 b 之間4或9的倍數（一列十個、欄寬4、靠左對齊），以及倍數個數與總和。

## 範例輸入
```
1
36
```

## 輸出結果
```
4   8   9   12  16  18  20  24  27  28  
32  36  
12
234
```

## 程式碼
```python
a = int( input() )
b = int( input() )
total = 0
count = 0
for i in range( a, b + 1 ):
    if i % 4 == 0 or i % 9 == 0:
        count += 1
        print("{:<4d}".format(i), end="")
        if count % 10 == 0:
            print()
        total += i
print()
print(count)
print(total)
```

# PYA407 不定數迴圈-閏年判斷

**類別：第4類：進階控制流程**

## 題目說明
以不定數迴圈輸入西元年份，判斷是否為閏年（每四年一閏，每百年不閏，每四百年也一閏）。輸入-9999結束。

## 範例輸入
```
2000
2001
-9999
```

## 輸出結果
```
2000 is a leap year.
2001 is not a leap year.
```

## 程式碼
```python
year = eval(input())
while year!=-9999:
    if (year % 4 == 0 and year % 100 != 0) or year % 400 == 0:
        print('%d is a leap year.'%year)
    else :
        print('%d is not a leap year.'%year)
    year = eval(input())
```

# PYA907 詳細資料顯示

**類別：第9類：檔案與異常處理**

## 題目說明
讓使用者輸入檔名，顯示該檔案的行數、單字數（以空白隔開）及字元數（不含空白）。

## 範例輸入
```
（需提供 read.txt）
```

## 輸出結果
```
（程式統計檔案的行數、單字數、字元數並輸出。
輸出格式範例：
3 line(s)
12 word(s)
48 character(s)）
```

## 程式碼
```python
f_name = input()
f_line = f_word = f_char = 0
with open(f_name, 'r') as file:
    for line in file:
        f_line += 1
        f_word += len(line.split())
        f_char += sum([len(c) for c in line.split()])
print('%d line(s)' %f_line)
print('%d word(s)' %f_word)
print('%d character(s)' %f_char)
```

# PYA903 成績資料

**類別：第9類：檔案與異常處理**

## 題目說明
讓使用者輸入五個人的名字，附加（append）到 data.txt 的尾端，之後顯示整個檔案的內容。

## 範例輸入
```
Alice
Bob
Carol
Dave
Eve
```

## 輸出結果
```
（程式將5個名字附加到 data.txt，並顯示整個檔案內容。
輸出範例：
Append completed!
Content of "data.txt":
（原有內容）
Alice
Bob
Carol
Dave
Eve）
```

## 程式碼
```python
with open("data.txt","a+",encoding="utf-8") as file:
    for i in range(5):
        file.write("\n"+input())
    print("Append completed!")
    print("Content of \"data.txt\":")
    file.seek(0)
    print(file.read())
```

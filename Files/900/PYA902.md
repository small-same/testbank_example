# PYA902 資料加總

**類別：第9類：檔案與異常處理**

## 題目說明
讀取 read.txt 的內容（數字以空白分隔），將這些數字加總後輸出，讀取完成後關閉檔案。

## 範例輸入
```
（需提供 read.txt）
```

## 輸出結果
```
（程式讀取 read.txt 中以空白分隔的數字並輸出總和。
範例：若 read.txt 內容為 '10 20 30 40'，則輸出為 100）
```

## 程式碼
```python
f=open("read.txt")
w=f.read()
sp=w.split(" ")
total=0
for x in sp:
    total+=eval(x)
print(total)
```

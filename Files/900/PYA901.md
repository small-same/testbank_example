# PYA901 成績資料

**類別：第9類：檔案與異常處理**

## 題目說明
讓使用者輸入五筆資料（學生姓名 + 期末總分），寫入 write.txt（每筆一行），寫入完成後關閉檔案。

## 範例輸入
```
Alice 90
Bob 85
Carol 78
Dave 92
Eve 88
```

## 輸出結果
```
（程式將5筆資料寫入 write.txt，無標準輸出。
write.txt 內容範例：
Alice 90
Bob 85
Carol 78
Dave 92
Eve 88）
```

## 程式碼
```python
n = open('write.txt','w')
for i in range(5):
    n.write(input()+'
')
n.close()
```

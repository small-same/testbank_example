# PYA908 單字次數計算

**類別：第9類：檔案與異常處理**

## 題目說明
讓使用者輸入檔名及出現次數，輸出檔案中恰好出現指定次數的單字，依字母排序。

## 範例輸入
```
（需提供 read.txt）
```

## 輸出結果
```
（程式讀取檔案，輸出恰好出現指定次數的單字，依字母排序。
範例：若 read.txt 中 'the' 出現2次而輸入次數為2，則輸出包含 'the'）
```

## 程式碼
```python
fn,num=input(),eval(input())
f=open(fn,'r')
w=f.read()
sp1=w.split()
setsp1=sorted(set(sp1))
for i in setsp1:
    if(sp1.count(i) == num):
        print(i,end='\n')
```

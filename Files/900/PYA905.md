# PYA905 字串資料刪除

**類別：第9類：檔案與異常處理**

## 題目說明
讓使用者輸入檔案名稱和字串 s，顯示刪除前後的檔案內容，並將刪除後的結果存檔。

## 範例輸入
```
（需提供 data.txt）
```

## 輸出結果
```
（程式讀取指定檔案，顯示刪除前後的內容。
輸出格式：
=== Before the deletion
（原始內容）
=== After the deletion
（刪除指定字串後的內容）
```

## 程式碼
```python
f=input()
s=input()
with open(f,"r+",encoding="utf-8")as file:
    data=file.read()
    print("=== Before the deletion")
    print(data)
    print("=== After the deletion")
    data=data.replace(s,'')
    print(data)
    file.seek(0)
    file.write(data)
```

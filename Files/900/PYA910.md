## 題目說明
讀取 read.dat（UTF-8編碼），第一列為欄位名稱，第二列後為個人記錄，輸出內容並統計男生（1）和女生（0）人數。

## 範例輸入
```
（需提供 read.dat）
```

## 輸出結果
```
（程式讀取 read.dat，輸出每行資料後統計性別。
輸出格式範例：
（每行資料）
Number of males: 3
Number of females: 2）
```

## 程式碼
```python
M=0
F=0
f=open('read.dat','r')
w=f.readlines()
for i in w:
    print(i)
    sp=i.split()
    if(sp[2]=="1"):
        M+=1
    elif(sp[2]=="0"):
        F+=1
print("Number of males: {}".format(M))
print("Number of females: {}".format(F))
```

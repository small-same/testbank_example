# PYA906 字串資料取代

**類別：第9類：檔案與異常處理**

## 題目說明
讓使用者輸入檔名、字串 s1 和字串 s2，將檔案中的 s1 以 s2 取代，並顯示取代前後的內容。

## 範例輸入
```
（需提供 data.txt）
```

## 輸出結果
```
（程式讀取指定檔案，顯示取代前後的內容。
輸出格式：
=== Before the replacement
（原始內容）
=== After the replacement
（取代後的內容）
```

## 程式碼
```python
fn,s1,s2=input(),input(),input()
f=open(fn,'r')
w=f.read()
print("=== Before the replacement")
print(w)
w=w.replace(s1,s2)
print("=== After the replacement")
print(w)
```

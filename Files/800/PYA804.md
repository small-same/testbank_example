# PYA804 大寫轉換

**類別：第8類：字串**

## 題目說明
讓使用者輸入一字串，分別將其轉換為全部大寫（upper）和每個字的第一個字母大寫（title）並輸出。

## 範例輸入
```
hello world
```

## 輸出結果
```
HELLO WORLD
Hello World
```

## 程式碼
```python
n=input()
print(n.upper())
print(n.title())
```

### 解法 2
```python
n=input()
res = ""
for i in n:
    if(i!=' ') :
        temp=ord(i)-32
        res = res + chr(temp)
    else :
        res +=' '
print(res)
print(n[0].upper(),end='')
for i in range(1,len(n)):
    if(n[i-1]==' '):
        temp=ord(n[i])-32
        print(chr(temp),end='')
    elif n[i]==' ' : print(' ',end='')
    else :
        print(n[i],end='')
```

# PYA809 密碼規則

**類別：第8類：字串**

## 題目說明
讓使用者輸入密碼，判斷是否符合規則（至少8字元、只含英文字母和數字、至少一個大寫字母），符合輸出 Valid password，否則輸出 Invalid password。

## 範例輸入
```
Password1
```

## 輸出結果
```
Valid password
```

## 程式碼
```python
n = input()
b=0
for i in range(len(n)):
    if n[i].isupper(): b=1
if b==1 and len(n)>=8 and n.isalnum(): print("Valid password")
else: print("Invalid password")
```

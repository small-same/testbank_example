# PYA807 字串加總

**類別：第8類：字串**

## 題目說明
讓使用者輸入一串由空白隔開的五個數字，計算並輸出其總和（Total）與平均（Average）。

## 範例輸入
```
10 20 30 40 50
```

## 輸出結果
```
Total = 150
Average = 30.0
```

## 程式碼
```python
m=list(map(int,input().split()))
j=sum=0
for i in m:
    sum+=i
    j+=1
print(f'Total = {sum}')
print(f'Average = {sum/j}')
```

# PYA810 最大值與最小值之差

**類別：第8類：字串**

## 題目說明
讓使用者輸入正整數 k（1≤k≤100），每筆測試資料為一串以空白隔開的數字，找出最大值和最小值之差，輸出至小數點後第二位。

## 範例輸入
```
2
1.5 2.3 8.7 4.1
10.0 2.5 7.3
```

## 輸出結果
```
7.20
7.50
```

## 程式碼
```python
for i in range(int(input())):
    data=list(map(float,input().split()))
    print('{:.2f}'.format( max( data ) - min( data )))
```

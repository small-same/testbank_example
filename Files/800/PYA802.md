# PYA802 字元對應

**類別：第8類：字串**

## 題目說明
讓使用者輸入一字串，顯示每個字元的ASCII碼及所有ASCII碼的總和。

## 範例輸入
```
Hi
```

## 輸出結果
```
ASCII code for +n[i]+ is 72
ASCII code for +n[i]+ is 105
177
```

## 程式碼
```python
n=str(input(''))
sum=0
for i in range(0,len(n)):
    sum+=ord(n[i])
    print('ASCII code for ''+n[i]+'' is '+str(ord(n[i])))
print(sum)
```

# PYA806 字元次數計算

**類別：第8類：字串**

## 題目說明
讓使用者輸入一字串和一字元，呼叫 compute() 函式回傳該字元在字串中出現的次數並輸出。

## 範例輸入
```
hello world
l
```

## 輸出結果
```
l occurs 3 time(s)
```

## 程式碼
```python
def compute(Str, s): print(f'{s} occurs {Str.count(s)} time(s)')
compute(input(),input())
```

### 解法 2
```python
def compute(Str, s):
    return Str.count(s)
Str = input()
s = input()
print(f'{s} occurs {compute(Str, s)} time(s)')
```

### 解法 3
```python
def compute(n,m) :
    ans=0
    for i in range(len(n)):
        if(n[i]==m): ans+=1
    print(f'{m} occurs {ans} time(s)')
compute(input(),input())
```

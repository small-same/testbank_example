# PYA801 字串索引

**類別：第8類：字串**

## 題目說明
讓使用者輸入一字串，顯示該字串每個字元的索引。

## 範例輸入
```
Hello
```

## 輸出結果
```
Index of +n[i]+: 0
Index of +n[i]+: 1
Index of +n[i]+: 2
Index of +n[i]+: 3
Index of +n[i]+: 4
```

## 程式碼
```python
n=str(input(''))
for i in range(0,len(n)):
    print('Index of ''+n[i]+'': '+str(i))
```

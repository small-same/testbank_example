# PYA210 三角形判斷

**類別：第2類：選擇敘述**

## 題目說明
讓使用者輸入三個邊長，檢查是否可組成三角形（任兩邊之和大於第三邊）。若可以輸出周長，否則輸出 Invalid。

> ⚠️ 原始程式碼 `print(Invalid)` 有 bug（缺少引號），已修正為 `print('Invalid')`。

## 範例輸入
```
3
4
5
```

## 輸出結果
```
12
```

## 程式碼
```python
a=int(input())
b=int(input())
c=int(input())
if(a+b>c and a+c>b and a+b>c): print(a+b+c)
else : print('Invalid')
```

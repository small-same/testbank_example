# PYA208 十進位換算

**類別：第2類：選擇敘述**

## 題目說明
讓使用者輸入一個十進位整數 num（0≤num≤15），將其轉換成十六進位值輸出。

## 範例輸入
```
13
```

## 輸出結果
```
D
```

## 程式碼
```python
n=eval(input())
if(n>=0 and n<10) : print(n)
elif (n==10) : print('A')
elif (n==11) : print('B')
elif (n==12) : print('C')
elif (n==13) : print('D')
elif (n==14) : print('E')
elif (n==15) : print('F')
```

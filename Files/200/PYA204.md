# PYA204 算術運算

**類別：第2類：選擇敘述**

## 題目說明
讓使用者輸入兩個整數 a、b 和一個算術運算子（+、-、*、/、//、%），輸出運算結果。

## 範例輸入
```
10
3
+
```

## 輸出結果
```
13
```

## 程式碼
```python
a=eval(input())
b=eval(input())
c=str(input())
if(c=='+'): print(a+b)
elif(c=='-'): print(a-b)
elif(c=='*'): print(a*b)
elif(c=='/'): print(a/b)
elif(c=='//'): print(a//b)
elif(c=='%'): print(a%b)
```

# PYA207 折扣方案

**類別：第2類：選擇敘述**

## 題目說明
讓使用者輸入購物金額（需≥8000），依折扣方案（8000起9.5折，18000起9折，28000起8折，38000起7折）輸出實付金額。

## 範例輸入
```
20000
```

## 輸出結果
```
18000.0
```

## 程式碼
```python
n=int(input())
if(n>=8000 and n<18000) : print(n*0.95)
elif(n>=18000 and n<28000): print(n*0.9)
elif(n>=28000 and n<38000): print(n*0.8)
elif(n>=38000): print(n*0.7)
```

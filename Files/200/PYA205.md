# PYA205 字元判斷

**類別：第2類：選擇敘述**

## 題目說明
讓使用者輸入一個字元，判斷它是英文字母（alphabet）、數字（number）或其它字元（symbol）。

## 範例輸入
```
a
```

## 輸出結果
```
a is an alphabet.
```

## 程式碼
```python
n=str(input())
if(n.isdigit()): print(f'{n} is a number.')
elif(n.isalpha()) : print(f'{n} is an alphabet.')
else : print(f'{n} is a symbol.')
```
